PROJECT PORTAL RULES ========================================:

- Пользователю портала дать только право 'Портал', никаких
    стандартных прав пользователей Odoo НЕ давать.

- В карточке пользователя портала указать все компании,
    по которым могут быть Проекты/Задачи/Вопросы для него.

- В Задачах/Вопросах, которые должны быть видны пользователям
    портала нужно указывать Проект.
